ProtoArc T1 Plus — User guide

Read this page in any browser on Mac, iPhone, iPad, Android, or desktop web. The touchpad driver itself runs only on macOS; other devices can use this guide for reference or sharing.

Platforms
macOS — full app iOS / iPadOS — this guide in Safari Android — Chrome / Firefox Web — any HTML viewer

1. What the macOS app does

ProtoArc T1 Plus is a menu bar app for Mac that connects to your ProtoArc T1 Plus touchpad over Bluetooth, reads touch data in user space, and moves the cursor, clicks, scrolls, and runs gesture shortcuts — without DriverKit or disabling SIP.

2. First-time setup (Mac)

  1. Open the app from your Applications folder (or Xcode build).
  2. Read the welcome slides, then enter your serial number (from SerialNumbers.txt).
  3. In System Settings → Privacy & Security, enable Input Monitoring and Accessibility for the app.
  4. Pair the touchpad via Bluetooth.
  5. Click the menu bar icon → Start Driver. Status turns green when the device is connected.
  6. Use Settings… for speeds, gestures, raw report log, and calibration.

3. How to use (daily)

  • Move — one finger on the pad.
  • Left click — quick one-finger tap.
  • Click and drag — tap, then hold and move (or use double-tap & slide if enabled in Settings).
  • Scroll — two fingers.
  • Right click — two-finger tap (touch-only).
  • Pinch — two-finger pinch for zoom shortcuts.
  • Three-finger swipes — Spaces, Mission Control, App Exposé (as configured).

4. Gesture overview

1 finger moveMove pointer
1 finger tapLeft click

3 finger swipe ↑Mission Control

Exact actions depend on your Settings toggles.

5. Tips

  • If the cursor misbehaves, try the Report layout picker in Settings and optional Log raw reports.
  • Seize device (when on) prevents macOS and the app from both handling the pad.
  • If the pointer feels “stuck” after using the menu bar, open the menu again briefly — the app clears drag state automatically.

6. iOS, Android, and web

There is no iOS or Android build of this driver: Apple and Google do not allow the same HID + accessibility event injection model for a third-party Bluetooth touchpad driver like on macOS.

Web — this file is plain HTML; host it on your site or AirDrop / cloud-open on any phone for the same instructions.
